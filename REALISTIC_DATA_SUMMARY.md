### 🎯 Usage Examples

#### Find all patients on Eylea
```python
from medications.models import Prescription
eylea_prescriptions = Prescription.objects.filter(instructions__icontains='Eylea')
for rx in eylea_prescriptions:
    print(f"{rx.patient.first_name} {rx.patient.last_name} - Batch: {rx.instructions.split('Batch: ')[-1]}")
```

#### Track OCT improvements for AMD patients
```python
from conditions.models import PatientCondition
from eye_tests.models import OCTScan

amd_patients = PatientCondition.objects.filter(condition__code='AMD')
for pc in amd_patients:
    scans = OCTScan.objects.filter(patient=pc.patient).order_by('test_date')
    first = scans.first()
    last = scans.last()
    if first and last:
        improvement = first.right_central_thickness - last.right_central_thickness
        print(f"{pc.patient.first_name} {pc.patient.last_name}: {improvement}μm reduction")
```
